This project focuses on synthesizing a **ZnTe/ZnMnS core-shell system** with the aim of shifting ZnTe core absorption from roughly **350 nm toward 400 nm** while still preserving successful **ZnMnS shell growth**.

The challenge is not only to move the optical signature of the core, but to do so without disrupting the interfacial chemistry required for robust shell formation.

To address that, the work explores **precursor combinations**, **solvent environments**, and **nucleation-growth conditions** in a systematic way.
